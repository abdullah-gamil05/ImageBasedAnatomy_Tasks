# Changelog

## [1.0.0] - 2025-01-05
### Added
- Image loading and viewing capabilities.
- Zoom functionality with multiple interpolation methods (nearest, linear, bilinear, cubic).
- Brightness and contrast adjustment with interactive sliders.
- Noise addition and denoising tools.
- Histogram visualization and SNR/CNR analysis.
- Undo and reset functionality.

### Fixed
- Minor GUI alignment issues.
- Improved error handling for unsupported image formats.

### Removed
- Deprecated legacy filter logic.
